## Running locally

First, run the development server:

```bash
pnpm install

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Your app should now be running on [localhost:3000](http://localhost:3000/).

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
